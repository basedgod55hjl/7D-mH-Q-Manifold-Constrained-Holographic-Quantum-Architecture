## Vision

ABRASAX GOD OS is a **never-before-seen AI stack** for all devices: a living, injectable operating layer that:

- Runs 7D Crystal programs (`.7d`, `.7dbin`)
- Integrates LM Studio for local LLM inference
- Provides a floating, draggable control panel UI
- Supports GGUF model loading and live fine-tuning
- Enables LLM-driven agent builders and dynamic OS injection

## File Extensions

- `.7d` – Source (7D-MHQL or translated C++)
- `.7dh` – Headers
- `.7ds` – Interface/headers
- `.7dbin` – Compiled binary
- `.7dso` – Shared library
- `.7dexe` – Executable
