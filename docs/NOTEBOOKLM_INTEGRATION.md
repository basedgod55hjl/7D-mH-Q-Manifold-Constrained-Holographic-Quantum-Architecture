## Adding Repo Docs to NotebookLM

1. **Create a Notebook**  
   - Go to [notebooklm.google.com](https://notebooklm.google.com)  
   - Create a new notebook

2. **Add Sources**
   - Use **Add source** → **Upload**
   - Add these docs:
     - `README.md`
     - `docs/LANGUAGE_SPEC.md`
     - `docs/ABRASAX_GOD_OS.md`
     - `docs/ARCHITECTURE.md`
     - `examples/mirrored_core/RUN_7D_GUIDE.md`

3. **MCP Integration**
   - With NotebookLM MCP in Cursor:
   - `research_start` – start research on a topic
   - `research_import` – import results into a notebook
   - `source_sync_drive` – sync Google Drive sources

4. **Suggested Notebooks**
   - **ABRASAX Core** – README, ABRASAX_GOD_OS, RUN_7D_GUIDE
   - **7D Language** – LANGUAGE_SPEC, Compiler_Architecture
   - **NVIDIA/AMD Docs** – External docs (upload PDFs or paste links)
